## 🏗️ 技术架构

### 技术栈
- **前端**: React 18 + TypeScript + Vite
- **桌面**: Electron
- **数据库**: SQLite + Prisma ORM
- **状态管理**: Zustand + React Query
- **样式**: Tailwind CSS + Headless UI
- **构建**: Turbo + pnpm

### 项目结构
```
minglog/
├── 📱 apps/
│   ├── web/              # React Web 应用
│   ├── desktop/          # Electron 桌面应用
│   └── api/              # Express API 服务器
├── 📦 packages/
│   ├── core/             # 核心业务逻辑
│   ├── database/         # 数据库操作层
│   ├── editor/           # 块编辑器组件
│   ├── ui/               # UI 组件库
│   └── search/           # 搜索引擎
├── 📚 docs/              # 项目文档
└── 🔧 配置文件           # 开发环境配置
```
