## 🎯 项目概况

**MingLog** 是一个基于模块化架构的all-in-one知识管理平台，采用本地优先设计理念，用户可根据需求自由选择和组合功能模块，创建个性化的知识工作空间。

### 🏗️ 技术架构优势
- **前端**: React 18 + TypeScript 5.5.4 + Vite 6.0.0
- **后端**: Rust + Tauri + SQLite + sqlx 0.7.3
- **架构**: 模块化设计 + 事件驱动 + 本地优先
- **性能**: <2秒启动，<200MB内存，<100ms渲染
