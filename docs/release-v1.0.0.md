## 🔧 已知问题和限制

### 当前限制
1. **WebDAV同步**: 接口已预留，完整实现计划在v1.1.0
2. **移动端应用**: 当前仅支持桌面端，移动端计划在v2.0.0
3. **协作功能**: 多用户协作功能计划在v1.2.0

### 性能注意事项
1. **大数据量**: 建议单个页面内容不超过10MB
2. **搜索性能**: 大量数据时建议使用更具体的搜索词
3. **内存使用**: 长时间使用建议定期重启应用
